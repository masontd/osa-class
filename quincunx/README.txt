README for Project 3

How to run the Galton (quincunx) simulation (2 ways):

1. python3 quincunx.py
    -defaults: Beans: 1000, bins: 11, start: 5, prob: 0.5

2. python3 quincunx.py --beans [int] --bins [int] --start [int] --prob [float]
    -you choose the number of beans, bins, start, and probability
